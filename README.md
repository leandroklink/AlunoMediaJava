# Projeto Aluno Média (Java)

Projeto simples em Java criado para praticar conceitos básicos de Programação Orientada a Objetos.

## Conceitos praticados

- Criação de classes e objetos
- Encapsulamento (`private`)
- Construtor e uso do `this`
- Classe imutável (`final`, sem setters)
- Sobrecarga de construtores
- Métodos (`calcularMedia`, `verificarSituacao`)
- Uso de `ArrayList<Aluno>`
- Entrada de dados com `Scanner`
- Tratamento do buffer do `Scanner` (`nextLine()`)

## Regras de negócio

- A média do aluno é calculada com duas notas
- Média maior ou igual a 7 → Aprovado
- Média menor que 7 → Reprovado
